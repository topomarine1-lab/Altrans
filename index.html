❓ Aide

🎯 Utilisation Correcte

1. Source : Choisir le système EPSG du point (ex: ED50/UTM 34N = 23034)

2. Saisir X, Y, Z dans les unités du système source

3. Cible : Choisir le système EPSG de destination (ex: WGS84/UTM 34N = 32634)

4. Cliquer TRANSFORM

⚠️ Important - EPSG différents

Pour avoir une vraie transformation, les EPSG source et cible doivent être DIFFÉRENTS :

  • WGS84 / UTM 34N = 32634
  • ED50 / UTM 34N = 23034 (différent !)
  • Carthage / UTM 32N = 22332
