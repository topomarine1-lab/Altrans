SubTopo-Geodetic-Coordinate Convers

Convertisseur de coordonnées géodésiques professionnel avec prise en charge de tous les systèmes mondiaux, import/export DXF, plan topographique avec cartouche, et base de données filtrable.


Fonctionnalités :

  • UTM zones 1-60 N/S (WGS84)
  • ED50, Carthage, IGN (Clarke 1880), et systèmes mondiaux
  • Calcul 7 paramètres Bursa-Wolf
  • Import/Export DXF avec transformation
  • Plan topographique avec cartouche professionnel
  • Export PDF, CSV, DXF
  • Base de données IndexedDB filtrable
  • Multilingue FR/EN/AR
