❓ Aide

🎯 Comment utiliser

  • 1. Source: EPSG + Datum + Coordonnées
  • 2. Cible: EPSG + Datum + Format souhaité
  • 3. Cliquer TRANSFORM

🇹🇳 Tunisie

  • Carthage UTM 32N: EPSG 22332
  • STT: EPSG 22392
  • NTT: EPSG 22391

⌨️ Raccourcis

  • Ctrl+N/O/S/P, F5 (Transform), F1 (Aide)
📖 Tutoriel

📚 Exemple Carthage → WGS84

1. Source EPSG: 22332
2. Preset "Carthage → WGS84"
3. Cible: WGS84 (4326)
4. TRANSFORM
